# A data set for determining strata areas (hectares)

Data necessary for determining the amount of area a strata covers to
expand estimates of biomass to be absolute. The data includes the
following columns:

- SUBAREA_ID

- MIN_DEPTH_M

- MAX_DEPTH_M

- AREA_HECTARES

- SUBAREA_SET_ID

- MIN_LAT_DD

- MAX_LAT_DD

## Usage

``` r
SA3
```

## Format

A data frame.
