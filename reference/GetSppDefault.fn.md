# Get Vector of Default Species Names for US West Coast

Get Vector of Default Species Names for US West Coast

## Usage

``` r
GetSppDefault.fn()
```

## Value

A vector of strings specifying common names with words separated using
underscores.

## Details

A standard list of common names for species of interest to the US West
Coast. This list does not reflect ecosystem or financial importance, and
instead it is strictly a list of species for which analyses take place.
